# Smart Customer Support Platform

A cloud-ready ticket management system designed using modern backend architecture principles.

## Tech Stack (Phase 1)
- Java
- Spring Boot
- PostgreSQL
- REST APIs

## Future Enhancements
- Microservices architecture
- Docker containerization
- AWS deployment
- Machine learning priority scoring
- Salesforce integration

## Project Goal
To simulate an enterprise-grade customer support backend system and progressively evolve it into a cloud-native microservices platform.
